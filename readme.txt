Step 1. " git clone https://github.com/Deeniedageenie/infs3208 && cd infs3208 " 
        to clone the docker-compose.yml and src files into the infs3208 directory.

Step 2. "docker-compose up -d"
        runs all the containers

Step 3. Go to http://{EXTERNAL IP}:8008 to check phpMyAdmin is running and import the database.sql into phpMyAdmin

Step 4. Go to http://{EXTERNAL IP}:8080/index.php to check the website is running.
